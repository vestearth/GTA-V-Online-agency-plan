`agents/docs` stores schema and reference material for agent-specific logic.

- Keep these files human-readable and stable.
- Do not duplicate runtime datasets here.
- `brand.schema.yaml` validates the Franklin brand catalog structure.
- `franklin.yaml` is the vehicle-brand reference catalog used to enrich vehicle recommendations.
- `agent14-cayo.yaml` is the heist benchmark reference for Cayo Perico planning.
- `tony.yaml` defines Tony reference metadata, thresholds, and the runtime contract for the Nightclub goods dataset.
